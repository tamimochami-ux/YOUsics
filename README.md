# YOUsics
YOUsics PRO est une application web de musique qui permet d’importer, écouter et organiser ses fichiers audio dans des playlists. Elle inclut un lecteur intégré avec barre de progression, un mode plein écran et un système de compte local pour sauvegarder les données. L’interface est simple, centrée et pensée pour une utilisation rapide et fluide.
